# Arduino-EEPROM-I2C

Use Arduino to control I2C EEPROM (512K)

You can overclock the I2C speed by modified the constant "TWI_FREQ" in <br /> twi.h under C:\Program Files (x86)\Arduino\hardware\arduino\avr\libraries\Wire\utility
